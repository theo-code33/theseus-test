## Installation

```bash
git clone https://github.com/theo-code33/theseus-test.git
cd theseus-test
npm install
```

## Run Tests

```bash
# Run once time
npm run test

# Run in watch mode
npm run test:watch

# Run with coverage
npm run test:coverage
```

## Run functions (excluding worldOfCode exercice)

```bash
cd path/to/exercice
node starter.js
```

## Run worldOfCode exercice

```bash
cd worldOfCode
npm install
npm run start
```
